# Termux (Android)

Bone does not currently support Termux or Android. The Bun-only CLI and its
native terminal renderer are released for macOS, Linux, and Windows on the
architectures listed in [GitHub Releases](https://github.com/frelion/bone/releases).

Running the package through Node.js in Termux is no longer a supported fallback.
Android support requires official Bun and native renderer release artifacts for
that platform before it can be added to Bone's support matrix.
