# Keyboard Shortcuts

Bone uses fixed built-in keyboard shortcuts. They are not configurable. Existing `~/.bone/agent/keybindings.json` files are ignored and left untouched.

## Editor

| Shortcut | Action |
|----------|--------|
| `Enter` | Submit input |
| `Shift+Enter`, `Ctrl+J` | Insert a newline |
| `Tab` | Complete a path or accept autocomplete |
| Arrow keys, `Ctrl+B`, `Ctrl+F` | Move the cursor |
| `Alt+Left`, `Ctrl+Left`, `Alt+B` | Move one word left |
| `Alt+Right`, `Ctrl+Right`, `Alt+F` | Move one word right |
| `Home`, `Ctrl+A` | Move to line start |
| `End`, `Ctrl+E` | Move to line end |
| `Backspace` | Delete backward |
| `Delete` | Delete forward |
| `Ctrl+W`, `Alt+Backspace` | Delete the previous word |
| `Alt+D`, `Alt+Delete` | Delete the next word |
| `Ctrl+U` | Delete to line start |
| `Ctrl+K` | Delete to line end |
| `Ctrl+Y`, `Alt+Y` | Yank and cycle deleted text |
| `Ctrl+-` | Undo |

## Application

| Shortcut | Action |
|----------|--------|
| `Escape` | Cancel autocomplete or abort the current operation |
| `Ctrl+C` | Copy a selection, clear the editor, or cancel a selector according to context |
| `Ctrl+D` | Exit when the editor is empty |
| `PageUp`, `PageDown` | Scroll chat history |
| `Shift+Left`, `Shift+Right` | Move focus between panes |

Selectors and conversation-management views show their fixed context-specific shortcuts in the interface.
